# Wild Animal Detection and Alert System using Vision Transformer

## Project Overview
This project aims to detect wild animals using a Vision Transformer (ViT) model and provide alerts to prevent human-wildlife conflicts.

## Objectives
- Detect animals from images
- Classify animal categories
- Improve safety in rural and forest areas
- Provide intelligent alert system

## Technologies Used
- Python
- Google Colab
- Vision Transformer (ViT)
- PyTorch / TensorFlow

## 📂 Project Structure
- `wild_animal_detection.ipynb` → Model training notebook

## Current Status
- Model training in progress
- Dataset preprocessing ongoing

## Future Improvements

- GUI interface
- Confidence score display
